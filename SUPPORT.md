# Support

## Start here

Read the repo's `AGENTS.md`. It documents how the repo works, what the
non-obvious rules are, and how to run things locally. Most questions are
answered there.

## Still stuck?

Open a [GitHub issue](../../issues/new/choose) in the relevant repo.
Use the bug report template for broken behaviour and the feature request
template for proposals.

## Security issues

See [SECURITY.md](SECURITY.md). Do not file security issues publicly.
